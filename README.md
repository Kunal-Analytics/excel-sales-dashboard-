# Sales Analysis Dashboard (Excel)

An interactive sales analytics dashboard built in Excel as a self-driven project to practice end-to-end data analysis — from raw data cleaning to a fully interactive, presentation-ready dashboard.

> ⚠️ **Disclaimer:** This project uses a practice/tutorial dataset styled with a real brand's look for learning purposes. It does **not** represent actual business data from that company, and this project is not affiliated with or endorsed by it. Product names (e.g., "Magnam Set," "Quia Gift") are placeholder values generated by the sample dataset itself.

## 🛠️ Tools & Skills Used

- **Power Query** — data cleaning and transformation
- **PivotTables** — core analysis and aggregation logic
- **Slicers** — interactive filtering by Order Date, Delivery Date, and Occasion
- **Charts** — bar and line visualizations
- **Dashboard Design** — KPI cards, grid layout, conditional formatting

## 💡 Insights

- Revenue shows two clear seasonal peaks (Feb–Mar and Aug) that line up closely with Holi and Raksha Bandhan — and "Colors" is the single highest-revenue product category, consistent with a Holi-driven spike.
- Anniversary gifting generates more revenue than Diwali or Valentine's Day in this dataset.
- Order volume is led by tier-2 cities (Imphal, Dhanbad, Kavali) rather than major metros.
- Most orders are placed in the evening, with activity peaking around 8 PM.

## 📁 How to Use

1. Download the `.xlsx` file from this repo
2. Open it in Excel (2016 or later recommended, for full Slicer/PivotTable support)
3. Use the slicers on the right-hand panel to filter the dashboard by Order Date, Delivery Date, or Occasion
